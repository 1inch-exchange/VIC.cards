# VIC.cards
«Very Important Contact Cards» to prioritise you communications.

# Install
``yarn``

# Build
``yarn run build``

# Run dev
``yarn run start``

## Used dependencies

* VCard: https://github.com/enesser/vCards-js
* QR Code: https://github.com/soldair/node-qrcode
